📈 Forecasting demand in Python(Holt Winter Method)

I developed a quarterly demand forecast for the 2025–2030 period using the Holt-Winters exponential smoothing method. The model captured both trend and seasonality patterns in historical data. With a MAPE (Mean Absolute Percentage Error) of 7.06%, the model demonstrated high accuracy, making it suitable for medium-term business planning and inventory decisions. Visualization and model evaluation were done using Python libraries such as statsmodels, pandas, and matplotlib.

📈 Food Detection using Yol0v5 (Computer Vision Project)

This project aimed to compare the performance of multiple YOLOv5 model variants — YOLOv5s, YOLOv5m, YOLOv5n, and YOLOv5x — in detecting everyday food items. The dataset consisted of 223 images collected from online sources, which underwent preprocessing, model training, image detection, and evaluation stages. The experiment used a clear split for training, validation, and testing. Results showed that YOLOv5s achieved the fastest training time and the highest mAP@0.5 score of 0.848, indicating a strong balance between speed and accuracy for lightweight food detection applications.

📈 Sentiment Analysis on Online Gambling Discussions using Chi-Square and Time Series

This project analyzed public sentiment toward online gambling (judi online) on platform X (formerly Twitter). Using text preprocessing techniques and a Chi-Square statistical method, we selected key features that influenced sentiment classification. Tweets were categorized into positive, negative, and neutral sentiments.

In addition, we applied time series analysis to observe the trend of each sentiment category over time. The model revealed important shifts in public perception — including spikes in negative sentiment following regulatory events.

📈 Spatial Regression of Online Gambling in Indonesia (GWR)

This project analyzes the influence of socio-economic factors on the number of active online loan users across 34 Indonesian provinces. Using Geographically Weighted Regression (GWR), it was found that unemployment affects 85.29% of provinces and average monthly expenditure affects 50%, highlighting the spatial variation in loan dependency across the country.
